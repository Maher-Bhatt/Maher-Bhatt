# 💫 About Me:

I'm a B.Tech CSE student at ITM (SLS) Baroda University (2025–2029) and co-founder of **[Velocity Web](https://velocityweb.in)** — a custom web development agency I run with two partners. I handle the technical side: architecture, frontend build, and delivery.

I'm frontend-strong. React, TypeScript, Vite, and Tailwind CSS are my daily stack. I've shipped real projects — a full-stack study platform (ITM Notes), an e-commerce site (Z-Tees), an NFT-based certificate system (CertChain, built for a hackathon), and client work through the agency. I completed a Python Development internship at QSkill, and I actively compete in cybersecurity CTFs — placed **Top 1,000 at Hack AI Thon** and **4th in CTF at Security BSides Vadodara**.

I'm early in my journey, and I know it. I don't pretend to know things I haven't built. What I do know: how to learn fast, how to ship under pressure, and how to execute cleanly when it matters.

Right now I'm focused on growing Velocity Web, sharpening my backend skills, and building things that actually solve problems — not just things that look good in a portfolio.
